## Why did I choose this project?
I found it really interesting that such incredible and hidden virtual world exists and wanted to explore it myself. The Dark Web is mostly associated with crimes, secret group and illegal activities, however I wanted to show that anyone can use it safely without fear and in an efficient way, which means actually finding something, despite the nature of this place, where anything real or safe is much harder to find then it seems to.
Another thing is the networking - I wanted to learn how the dark web manages to keep itself so secure and private, how is it different while compared to Internet we all use daily.
